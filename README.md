# StixNVibes

A modern sticker store web application built with React and Vite.

## 🌟 What's New?

### 🧊 Glass UI with Refraction (Aceternity Inspired)
The landing page has been completely overhauled from its original brutalist aesthetic to a sleek, modern **Glass UI**. This includes:
- Deep `backdrop-blur-2xl` glass panels for content structure.
- Animated, abstract gradient refraction "blobs" moving smoothly in the background.
- High-contrast, glowing typography with vibrant pinks, purples, and blues.
- Custom shimmer animations on primary call-to-action buttons.

### 🎨 Key Features
- **Responsive Design:** Works flawlessly on desktop and mobile.
- **Glassmorphism Components:** Implemented using pure Tailwind CSS and Framer Motion.
- **Dynamic Cart System:** Real-time floating cart with WhatsApp checkout integration.
- **Modular Data Structure:** Easily modifiable JSON files for products and collections.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Build for Production
```bash
npm run build
```
